CONTEXT API-
------------
Manlo mein hu Main page pe
Man lete hai Main Page ke andar ek section hai Header ka section mtlb ek aur page chla gya Header
Mtlb Main Page se Header page mein data gaya.
Ab manlo about Header ke andar ek section hai Navbar ka mtlb ek aur page chla gya Navbar.
Mtlb Main Page se Header page mein data gaya aur Header page se data gya Navbar ke page mein 
Main Page --> Header --> Navbar

STEPS-
-------
1. Page.js file mein rafce command likho
2. App folder ke andar ek Components folder banao
3. Components folder mein Header.js file banao.
4. Header.js file mein rafce likho.
5. Main Page.j file mein useState import aur variable banao num aur uski value 10 rakhdo.
6. Main Page.js file mein <Header num={num} /> Component ke sath variable bhejhna hai toh yeh likho
7. Fir apne Header.js file mein jao aur as a prop Header ke function mein vo object recive karo.
8. Ab ap yaha Homepage se Header mein data leke aa pa rhe ho, lekin mujhe Navbar mein bhejhna hai toh?
8. Components folder mein Navbar.js file banao.
9. Navbar.js file mein rafce likho.
10. Fir apne Navbar.js file mein jao aur as a prop Navbar ke function mein vo object recive karo.

    ---------------------
    Mtlb data sabhi ko provide karna hai through Context
    <Mycontext.Provider>
     {children}  -->  isme children likhdo mtlb
    </ Mycontext.Provider> 
    Aur context ke arrow function mein bhi {children} as params bhejho
