TWO WAY BIDNING- ap directly change mat karo dom mein react ko bolo vo krdega.
Steps-
------
1. Create a form
2. Inside form create a input field give it a border to see.
3. Ab mein kuch bhi input field mein changes krunga toh mein directly kar pa rha hu but mujhe yeh nhi krna.
4. Ap yeh changes react ko bolo vo krega.
5. Ek variable banao jisko pehle empty rakho aur uski value us input field mein maintain rakho using that variable.
6. Fir onChange pr ek callback function call kro jisme event milega usey kuch value milegi usmein useState vale function mein vo value dalo.

CODE -
-----------------
'use client'
import React, { useState } from 'react'

const page = () => {
  const [username, setusername] = useState('')
  return (
    <>
    <h1 className='mb-5 text-xxl'>Enter your name:</h1>
    <form>
      <input type='text' className='border-2 border-zinc-800 px-4 py-2 text-xl' value={username} onChange={(e)=>{
        setusername(e.target.value)
        console.log(username);
      }}></input>
    </form>
    </>
  )
}

export default page

Routing , Dynamic Routing -
----------------------------
Steps-
------
1. Create folder with the same name for the page that you want to navigate inside that create a page.js file.
2. use command rafce to create component for that page.
3. Now in main page.js file inside your main div or fragement element, create a anchor tag and give href=/FolderName
4. Now when you click on Contact anchor tag you will be navigate to Contact page.

Main page.js file Code-
-----------------------
import React from 'react'

import Link from 'next/link'; --> import link also

const page = () => {
  return (
    <div>
      <h1 className='text-2xl font-bold'>This is Home Page</h1>
      {/* <a href='/Contact'>Contact</a> */} --> Reload hoga page
      <Link href='/Contact'>Go to Contact Page</Link> --> Reload nahi hoga page
    </div>
  )
}

export default page

'use client'
import axios from 'axios'
import React, { useState } from 'react'

const page = () => {

  const [users, setusers] = useState([])

  const getUsers = async () => {
    const res = await axios.get("https://jsonplaceholder.typicode.com/users")
    const data = res.data
    console.log(data);
    setusers(data);
  }

  return (
    <>
    <button onClick={getUsers} className='bg-green-600 text-white px-4 py-2 rounded font-bold'>Get Data</button>
    <div className='p-8 bg-slate-100 mt-5'>
      <ul>
        {users.map((e)=>{
          return <li>{e.name}</li>
        })}
      </ul>
    </div>
    </>
  )
}

export default page
